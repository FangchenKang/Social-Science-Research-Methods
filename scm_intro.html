合成控制法：当没有现成对照组时，如何“造”一个反事实

合成控制法的核心不是复杂公式，而是一个非常直观的反事实逻辑：用多个未处理地区按权重组合，合成一个在政策前最像处理地区的“虚拟对照组”，再用政策后的差距识别政策效应。

一、SCM 要解决的问题：处理对象只有一个，普通对照组不够像

合成控制法最常用于一个地区、一个城市、一个国家或少数几个单位受到政策冲击的研究场景。比如某个城市率先设立数据局，某个省份率先推行数字政府改革，某个区县率先成为国家级试点。研究者想知道政策是否有效，但真正困难的是：这个处理地区如果没有政策，本来会怎样。

普通 DID 通常需要找到一组真实对照组，并假设处理组和对照组在没有政策时具有平行趋势。但如果处理对象非常特殊，单独找一个对照地区往往不合适。一个城市可能经济基础、人口规模、产业结构、治理能力都很特殊，任何单一城市都不像它。SCM 的办法不是强行找一个对照对象，而是用多个未处理对象加权组合，合成一个更像它的反事实对照。

处理对象Treated Unit

受到政策冲击的城市、省份、县区、国家或组织。

供体池Donor Pool

没有受到政策影响、可用于合成对照的若干单位。

合成对照Synthetic Control

供体池中多个单位的加权平均，用来模拟处理对象的反事实。

二、核心直觉：政策前像，政策后差，差距才可能是政策效应

SCM 的关键判断是：如果合成对照在政策发生前能够很好地复刻处理对象的趋势，那么它就可以作为处理对象“没有政策时”的反事实近似。政策发生后，处理对象的真实结果与合成对照之间出现的差距，就可以被解释为政策的估计效应。

真实处理地区

政策前有一条实际发展轨迹，政策后受到政策影响，走向真实结果。

＋
多个未处理地区

按权重组合，尽量让政策前走势复制处理地区，形成“合成版本”。

政策效应 = 处理地区政策后的真实结果 − 合成对照政策后的预测结果

这个公式背后不是简单相减，而是反事实比较：合成对照代表处理地区如果没有接受政策时可能走出的路径。

四、权重怎么来：不是任意平均，而是让政策前最像

SCM 的权重不是研究者主观拍脑袋决定的，而是通过优化得到的。它会在供体池中寻找一组非负权重，并要求所有权重加总为 1，使得合成对照在政策前的结果轨迹和重要预测变量上尽可能接近处理对象。

选择权重 W = (w₁, w₂, ..., wⱼ) 满足：wⱼ ≥ 0，且 Σwⱼ = 1 目标：让政策前的处理地区 ≈ Σ wⱼ × 供体地区ⱼ 政策后效应：αₜ = Y₁ₜ − Σ wⱼYⱼₜ

这组权重具有一个很重要的含义：权重越高，说明该供体单位越有助于复制处理单位的政策前特征。权重为 0 的地区并不是被删除，而是说明它对构造这个反事实没有实质帮助。

五、SCM 的识别条件：合成对照必须是可信反事实

SCM 的说服力来自政策前拟合和供体池质量。如果合成对照在政策前就无法复制处理地区，那么政策后差距就不能被解释为政策效应；如果供体池中的单位受到政策溢出影响，那么合成对照也不再是未处理状态。

政策前拟合好

合成对照在政策发生前应当尽量贴近处理地区的结果轨迹和核心预测变量。

供体池干净

供体单位不能接受同类政策，也不能明显受到处理地区政策的溢出影响。

没有提前反应

处理地区不能在政策正式发生前已经因预期政策而改变行为或结果。

凸包问题

如果处理地区太特殊，所有供体加权后也不像它，SCM 就难以构造可信对照。

局部解释

SCM 解释的是该处理地区的政策效应，不一定能推广到所有地区。

坏供体池

把明显不相关、被政策污染或制度环境完全不同的单位放入供体池，会削弱识别。

六、如何判断结果可信：看拟合、看安慰剂、看差距是否异常

SCM 通常不能像大样本回归那样只依赖显著性星号。它更强调图形证据、政策前拟合、安慰剂检验和相对排名。如果处理地区政策后的差距明显大于大多数安慰剂地区，政策解释才更有说服力。

1. 政策前拟合检验

政策前处理地区和合成对照越接近，说明合成对照越能代表反事实。如果政策前差距就很大，政策后的差距就不能轻易归因于政策。

2. 空间安慰剂检验

把供体池中的每个未处理地区轮流假装成处理地区，分别构造合成对照。如果真实处理地区的政策后差距远大于这些假处理地区，说明结果不太可能只是偶然波动。

3. 时间安慰剂检验

把政策时间提前到一个并没有真正发生政策的年份。如果假政策时间也出现明显效应，说明结果可能来自已有趋势，而不是政策本身。

4. 供体池稳健性检验

删除某些高权重供体、调整供体池范围或改变预测变量后，如果结论仍然稳定，说明结果不依赖某一个特殊供体或特殊设定。

七、一篇 SCM 论文通常怎么做

SCM 的写作重点不是“我用了合成控制法”，而是说明为什么这个政策适合用 SCM、供体池为什么合理、合成对照为什么可信、政策后差距为什么可以解释为政策效应。

1. 确定政策冲击一个地区或少数地区在明确时间点接受政策处理。
2. 构造供体池选择未受政策影响、制度环境可比的候选对照单位。
3. 选择预测变量使用政策前结果轨迹和影响结果的重要处理前变量。
4. 估计权重让合成对照在政策前尽量拟合处理单位。
5. 解释效应比较政策后真实结果和合成对照，并通过安慰剂与稳健性检验增强可信度。

八、自测：读懂 SCM 的三个关键问题

问题 1：SCM 和 DID 最大的不同是什么？

DID 通常依赖真实对照组和平行趋势，SCM 则用多个未处理单位加权合成一个虚拟对照组。SCM 尤其适合处理单位很少、难以找到单一对照组的情形。

问题 2：为什么政策前拟合如此重要？

因为合成对照要代表处理单位没有政策时的反事实。若政策前都不像，政策后差距就可能只是原本差异延续，而不是政策效应。

问题 3：SCM 的政策效应能否推广到所有地区？

通常不能。SCM 估计的是特定处理单位的政策效应，外部推广需要非常谨慎，除非有更多案例或额外理论支持。

一句话记住：SCM 的本质，是用多个未处理单位合成一个政策前最像处理单位的反事实，再用政策后的差距识别政策效应。
