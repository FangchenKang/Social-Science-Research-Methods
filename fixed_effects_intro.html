METHOD NOTE

固定效应模型：把“不变的差异”从因果比较中扣掉

固定效应模型的核心不是多加几个虚拟变量，而是在面板数据中用“自己和自己比”，控制那些看不见但长期不变的差异。

一、核心直觉：同一个对象在不同时点的变化，更适合用来识别影响

固定效应模型处理的是面板数据中的可比性问题。很多单位本来就不同，比如城市有长期发展基础差异，学校有历史声誉差异，个人有能力差异，企业有管理文化差异。如果这些差异既影响解释变量 X，也影响结果变量 Y，普通回归就会把“本来就不同”误当成“X 的影响”。

固定效应的底层想法是：不要简单拿 A 和 B 比，而是更多利用 A 自己随时间的变化来解释 A 自己结果的变化。
普通比较的问题

不同单位之间本来就不一样

数字治理水平高的城市，可能本来财政能力强、干部能力高、产业基础好。这些长期差异也会提高治理绩效，所以直接比较高数字治理城市和低数字治理城市，容易高估数字治理的作用。

固定效应的办法

把不随时间变化的差异扣掉

如果一个城市的地理位置、历史基础、长期行政传统在研究期内基本不变，固定效应就可以把这些稳定差异吸收掉。剩下的识别主要来自同一城市内部随时间发生的变化。

二、为什么需要固定效应：遗漏变量如果长期不变，就可以被吸收

固定效应模型最常见的应用场景，是存在不可观测但相对稳定的遗漏变量。比如研究财政透明度对公众信任的影响，地方政治文化可能同时影响透明度和信任，但政治文化很难准确测量。如果政治文化在研究期内变化不大，地区固定效应就能控制它。

普通模型：Y_it = βX_it + u_i + ε_it
问题：u_i 是看不见的单位特质，如果它和 X_it 相关，β 会有偏。

这里的 u_i 可以理解为每个单位长期不变的特质。它可能是城市历史基础、个人能力、企业文化、学校声誉、地区治理传统。固定效应模型把 u_i 单独拿出来控制，从而避免它污染 X 对 Y 的估计。

三、个体固定效应：控制每个单位不随时间变化的特质

个体固定效应的目标，是控制每个研究对象自身长期稳定的差异。这里的“个体”不一定是人，也可以是城市、省份、学校、企业、社区或国家。只要同一个单位被反复观察，就可以考虑个体固定效应。

Y_it = βX_it + α_i + ε_it

α_i 就是个体固定效应。它给每个单位一个自己的截距，相当于允许每个城市、每个企业、每个人有自己的基础水平。这样，模型不再要求所有单位从同一个起点出发，而是让每个单位带着自己的长期特质进入比较。

可以被个体固定效应控制的因素为什么
地理位置、历史基础、长期文化这些特征在短期内基本不变
个人天赋、家庭出身、早期教育背景如果研究对象是个人，这些因素通常稳定
企业长期管理风格、学校声誉研究期内变化较慢，可被单位固定效应吸收

四、时间固定效应：控制所有单位共同经历的时间冲击

时间固定效应用来控制某一年或某一时期所有单位共同受到的影响。比如宏观经济波动、全国性政策调整、疫情冲击、技术扩散、统一考核变化。这些冲击会同时影响很多单位，如果不控制，可能被误认为 X 的作用。

Y_it = βX_it + λ_t + ε_it

λ_t 就是时间固定效应。它给每一年或每一期一个共同截距，相当于允许所有单位在某一年一起上升或一起下降。这样，模型可以把宏观时间趋势和共同冲击扣掉。

五、双向固定效应：同时控制单位差异和时间冲击

双向固定效应模型是政策评估中最常见的形式。它同时加入个体固定效应和时间固定效应，既控制单位长期不变的差异，也控制所有单位共同经历的时间变化。

Y_it = βX_it + α_i + λ_t + ε_it

这个模型中的 β 来自哪里？它不是来自城市之间的简单横向差异，也不是来自所有年份共同上升下降的趋势，而是来自同一单位内部、扣除共同时间冲击之后，X 的变化与 Y 的变化之间的关系。

六、组内变换：固定效应其实是在看“偏离自身平均值的变化”

固定效应模型背后的数学直觉，是把每个单位自己的平均水平减掉。这样，那些对该单位而言长期不变的特质就消失了。模型比较的不是城市 A 和城市 B 谁更高，而是城市 A 在某一年是否高于自己的长期平均，以及这种偏离是否伴随结果变量的偏离。

Y_it - Ȳ_i = β(X_it - X̄_i) + (ε_it - ε̄_i)

这个式子说明，个体固定效应估计的 β 来自单位内部变化。比如一个城市的数字治理投入在某几年高于它自己的平均水平，治理绩效是否也高于它自己的平均水平。固定效应把“城市之间的长期差异”拿掉，保留“城市内部随时间变化”的信息。

固定效应不是让所有单位一样，而是让每个单位先和自己的长期平均比，再看变化之间是否同步。

七、互动演示：固定效应如何扣掉“本来就高”的差异

八、论文中应该怎么写

固定效应模型的写作应说明为什么存在不可观测但稳定的差异，以及为什么需要控制这些差异。比较好的表达是：为控制地区之间不随时间变化的异质性，模型加入地区固定效应；为控制所有地区共同经历的宏观时间冲击，模型加入年份固定效应。

Y_it = βX_it + γControls_it + α_i + λ_t + ε_it

报告结果时，要说明标准误如何处理。面板数据中，同一地区不同年份的误差项往往相关，因此常见做法是按地区聚类稳健标准误。若政策或处理在地区层面变化，通常也应按地区层面聚类。

九、常见误区：固定效应不是万能控制器

误区一：以为固定效应能控制一切

固定效应只能控制不随时间变化的遗漏变量。那些随时间变化、同时影响 X 和 Y 的因素仍然可能造成偏误。

误区二：忽略变量缺乏组内变化

如果一个变量在单位内部几乎不变化，固定效应模型就很难估计它的影响。完全不随时间变化的变量会被个体固定效应吸收掉。

误区三：把固定效应等同于因果识别

固定效应改善了可比性，但不自动保证因果关系。仍要考虑反向因果、时间变化遗漏变量、选择性政策实施等问题。

误区四：标准误不聚类

面板数据通常存在同一单位内的序列相关。若不做聚类稳健标准误，显著性可能被高估。

十、自测：固定效应到底控制了什么

问题 1：地区固定效应能否控制一个城市长期稳定的历史基础？
可以。只要历史基础在研究期内基本不变，地区固定效应就能吸收这类单位层面的稳定差异。
问题 2：地区固定效应能否控制某一年全国共同发生的疫情冲击？
不能单独控制。全国共同时间冲击需要年份固定效应来控制。
问题 3：如果一个解释变量在每个地区都不随时间变化，加入地区固定效应后还能估计它吗？
不能。完全不随时间变化的变量会被地区固定效应吸收，因为固定效应只利用单位内部随时间变化的信息。
一句话记住：固定效应模型用“自己和自己比”控制稳定差异，用时间固定效应扣除共同冲击，留下组内变化来识别关系。
