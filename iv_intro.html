因果识别方法 · IV 工具变量法

当 X 不干净，怎样识别 X 对 Y 的因果效应？

IV 的核心不是“找一个相关变量”，而是找一个只推动 X、但不绕过 X 直接影响 Y 的外部工具。它把 X 中相对外生的那部分变化提取出来，用这部分变化识别因果效应。

一句话讲清楚 IV

普通回归用 X 的全部变化解释 Y，但 X 可能混入能力、家庭、地区基础、反向因果等内生因素；IV 只用工具变量 Z 推动的那一部分 X 的变化来解释 Y。

普通回归：Y = α + βX + ε
但 X 可能与 ε 相关 → β 不可信

IV 思路：Z → X → Y
只使用 Z 带来的 X 的外生变化

判断 IV 是否可信，不看它和 Y 有没有相关，而看它是否只通过 X 影响 Y。

一、IV 要解决的不是相关问题，而是内生性问题

当解释变量 X 与误差项相关时，普通回归无法把 X 的因果效应从其他混杂因素中分离出来。IV 的出现，正是为了在 X “不干净”的情况下，寻找一部分相对外生的变化。

遗漏变量

看不见的因素同时影响 X 和 Y

比如能力、家庭背景同时影响教育年限和收入，教育变量就混入了能力与家庭资源。

反向因果

不是 X 影响 Y，也可能是 Y 影响 X

比如财政收入高的地方更可能建设数字平台，平台建设和治理绩效之间就可能互相影响。

测量误差

X 被测得不准，也会导致偏误

比如真实治理能力难以测量，只能用替代指标，指标噪音会削弱或扭曲估计结果。

教育与收入的经典直觉

如果直接用教育年限解释收入，得到的系数不一定是教育的真实回报。教育年限高的人，可能本来能力更强、家庭条件更好、成长环境更优，这些因素也会影响收入。IV 的办法是找一个外部因素 Z，例如义务教育改革、出生季度、学校距离等，用它带来的教育变化来估计教育对收入的影响。

二、工具变量的底层逻辑：Z 推动 X，但不直接推动 Y

一个好的工具变量像外部推力。它必须能改变核心解释变量 X，同时不能绕开 X 直接影响结果 Y。IV 因此不是随便找一个与 X 相关的变量，而是要找到一条可信的因果链。

允许的路径

Z 影响 X，X 再影响 Y。比如教育改革让一部分人多读书，多读书再影响收入。

危险的路径

Z 不能直接影响 Y，也不能通过其他渠道影响 Y。否则 IV 估计混入了别的因果机制。

三、IV 成立需要两个核心条件

相关性决定工具变量有没有力量，外生性决定工具变量是否干净。前者可以用数据检验，后者更多依赖制度背景、理论论证和排除替代解释。

第一阶段

相关性：Z 必须影响 X

如果工具变量不能显著改变核心解释变量，它就无法提供用于识别的外生变化。弱工具变量会让 IV 估计非常不稳定，甚至比普通回归更糟。

经验判断：第一阶段通常要报告 Z 对 X 的影响，并关注 F 统计量。F 值过低时，需要警惕弱工具变量问题。
排除限制

外生性：Z 只能通过 X 影响 Y

工具变量不能与遗漏变量相关，也不能绕过 X 直接影响 Y。比如“离大学远近”若同时代表城市化水平和就业机会，它就可能直接影响收入，而不只是影响教育。

外生性最难证明。它通常不是一个统计检验可以完全解决的问题，而要靠研究设计和制度事实说服读者。
条件问题不满足时的后果
相关性Z 是否真的推动 X？工具太弱，估计不稳定，标准误变大，结论不可信。
外生性Z 是否只通过 X 影响 Y？工具不干净，IV 估计仍然有内生性，甚至方向错误。

四、两阶段最小二乘：先提取 X 的外生部分，再解释 Y

2SLS 是 IV 最常见的估计方式。第一阶段用 Z 预测 X，第二阶段用预测出来的 X 解释 Y。它的本质不是数学技巧，而是把 X 中由工具变量推动的部分分离出来。

在最简单的二元工具变量情形中，IV 效应可以理解为“Z 对 Y 的影响”除以“Z 对 X 的影响”。

IV = ΔY / ΔX = (Z 对 Y 的影响) / (Z 对 X 的影响)

五、IV 估计的常常不是总体效应，而是 LATE

IV 识别的是那些会因为工具变量变化而改变 X 的人的效应。这个人群叫遵从者，估计结果通常被解释为局部平均处理效应。

遵从者

会被 Z 推动的人

政策改革让他们多读书，距离变化让他们更可能上大学。IV 主要识别这类人的效应。

总是处理者

不管 Z 怎样都会接受处理

无论有没有教育改革、离学校远近如何，他们都会继续接受教育。

从不处理者

不管 Z 怎样都不接受处理

工具变量变化也不会改变他们的处理状态，因此他们不贡献 IV 的识别效应。

LATE 的含义

如果用义务教育改革作为工具变量，IV 估计的不是所有人多读一年书的平均收益，而是那些因为改革才多读书的人，教育对他们收入的影响。如果用大学距离作为工具变量，估计的是那些因为离大学更近才上大学的人，大学教育对他们的影响。

六、判断 IV 是否可信，要重点看这些风险

IV 的难点不在二阶段回归，而在工具变量是否真正满足识别假设。下面这些问题，是阅读和写作 IV 论文时必须追问的地方。

弱工具变量

Z 对 X 的影响很弱，导致估计不稳定。第一阶段不强，第二阶段再漂亮也要谨慎。

违反排除限制

Z 除了通过 X，还通过其他渠道影响 Y。比如地理距离既影响教育，也影响就业市场。

工具变量与遗漏因素相关

Z 本身不是随机的，可能反映地区基础、家庭背景、制度资源或选择偏差。

单调性问题

工具变量应朝同一方向影响处理状态，不能让一部分人增加 X、另一部分人减少 X。

七、把 IV 写进论文时，应该讲清楚什么

一篇好的 IV 研究，不是把工具变量放进模型就结束，而是要完整说明工具变量为什么有效、为什么外生、估计对象是谁、结论能推广到哪里。

写作环节必须回答的问题
内生性来源为什么普通回归不可信？是遗漏变量、反向因果还是测量误差？
工具变量来源Z 来自什么制度、历史、地理或自然变化？为什么它能推动 X？
相关性检验第一阶段是否足够强？是否存在弱工具变量风险？
外生性论证Z 为什么不会直接影响 Y？为什么不与遗漏变量相关？
估计对象解释估计的是总体效应，还是遵从者的局部平均处理效应？
稳健性与替代解释有没有可能存在其他渠道？有没有做排除、安慰剂或机制检验？

八、自测：你真的理解 IV 了吗？

问题 1：一个变量只要和 X 相关，就可以作为工具变量吗？

问题 2：如果 Z 能直接影响 Y，即使它也影响 X，还能作为工具变量吗？

问题 3：IV 估计值通常可以无条件推广到所有人吗？

最后一句话

IV 的本质，是在 X 不干净时，借助一个外部工具 Z，把 X 中相对外生的变化提取出来，再用这部分变化识别 X 对 Y 的因果效应。它的核心不是二阶段公式，而是“Z 是否只通过 X 影响 Y”这一识别逻辑。
