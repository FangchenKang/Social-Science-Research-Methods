METHOD NOTE

事件研究：把政策效应拆成“发生前、发生时、发生后”的动态路径

事件研究的核心不是多画一张图，而是围绕政策发生时点重排时间，检验政策前是否可比，并观察政策后效应如何变化。

核心直觉相对时间模型形式平行趋势动态效应互动演示研究流程常见误区自测

一、核心直觉：不要只问“有没有效果”，还要问“效果什么时候出现”

事件研究，英文是 Event Study，常和 DID 放在一起使用。DID 通常给出一个平均政策效应，而事件研究把政策前后每一期的效应都估计出来。它既可以检验政策前处理组和对照组是否已有不同趋势，也可以观察政策实施后的效果是立即出现、逐步增强，还是短暂后消失。

一句话记住：DID 看平均差异，事件研究看动态路径；政策前看可比性，政策后看效应演化。
政策前

检验是否提前分化

如果政策实施前，处理组相对于对照组已经明显上升或下降，就说明平行趋势可能不成立，后续政策效应解释会变弱。

政策后

观察效应如何展开

政策影响可能不是一次性出现，而是逐步积累、滞后发生、短期冲击后回落，事件研究可以展示这种过程。

二、相对时间：把不同地区的政策发生点对齐

事件研究最重要的处理，是把日历时间转换为相对时间。政策发生那一年记为 0，政策前一年记为 -1，政策后一年记为 +1。这样，哪怕不同地区政策发生年份不同，也可以围绕各自的事件时点进行比较。

三、模型形式：用一组相对时间虚拟变量替代一个政策变量

普通 DID 常用一个 Treat×Post 表示政策发生后的平均处理效应。事件研究则把政策发生前后每一期都变成相对时间虚拟变量，再分别估计每一期的系数。

Y_it = α_i + λ_t + Σ_k β_k · 1[相对时间 = k] + ε_it
通常省略 k = -1，把政策前一期作为基准期。

β_k 的含义是：在第 k 个相对时期，处理组相对于基准期和对照组的变化幅度。政策前的 β_k 应该接近 0，政策后的 β_k 则展示政策效应的动态变化。

四、政策前系数：事件研究首先是平行趋势检验

事件研究最重要的功能之一，是观察政策前处理组和对照组是否已经存在系统性差异。如果政策前的系数明显不为 0，或者呈现持续上升、下降趋势，就说明处理组在政策前已经和对照组走向不同。

理想情况

政策前系数接近 0

这说明处理组和对照组在政策前没有明显分化，平行趋势更有说服力。

危险信号

政策前已有趋势

如果政策前系数已经明显上升，政策后的上升就可能不是政策造成的，而是原有趋势延续。

五、政策后系数：看政策效应是即时、滞后还是递增

事件研究的另一个价值，是展示政策效果的动态形态。制度改革可能有滞后效果，基础设施建设可能逐步增强，行政运动可能短期明显但很快衰减。

即时效应

政策发生当期或下一期立即出现明显变化。

滞后效应

政策实施后几期才逐渐出现影响。

持续效应

政策影响出现后保持稳定或继续扩大。

七、事件研究通常怎么做

第一步

确定事件时点

明确每个处理单位什么时候受到政策、改革、冲击或事件影响。

第二步

构造相对时间

把事件发生时点设为 0，向前向后生成 lead 和 lag 变量。

第三步

估计并画图

省略一个基准期，估计每个相对时期系数，并用图展示置信区间。

八、常见误区：事件研究不是平行趋势的万能证明

误区一：只看政策后显著

政策前系数同样重要。如果政策前已经显著变化，政策后系数再漂亮也要谨慎。

误区二：把“不显著”当作完全平行

政策前不显著只能说明没有发现明显差异，不等于平行趋势一定成立。

误区三：忽略提前反应

如果政策公布早于实施，处理组可能提前调整行为，导致政策前系数已经变化。

误区四：多期 DID 直接套传统 TWFE

如果政策分批实施，传统双向固定效应事件研究可能产生偏误，需要考虑现代分期处理方法。

九、自测

问题 1：事件研究图中政策前系数明显上升，说明什么？
说明处理组在政策前已经相对对照组发生变化，平行趋势可能不成立，政策效应解释要谨慎。
问题 2：为什么通常省略相对时间 -1？
因为需要选择一个基准期作为比较参照。常用政策前一期 -1 作为基准期，其他时期系数都相对于它解释。
问题 3：事件研究和 DID 的关系是什么？
事件研究可以理解为 DID 的动态扩展。DID 给出平均政策效应，事件研究展示政策前后每一期的动态效应。
一句话记住：事件研究围绕政策发生时点重排时间，政策前检验可比性，政策后观察效应演化。
