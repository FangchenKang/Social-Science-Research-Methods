METHOD NOTE

多层模型 HLM：把“个体嵌套在组织和地区中”放进模型

HLM 的核心不是多跑一层回归，而是承认数据有层级结构：个人属于社区，学生属于学校，居民属于城市。上层环境会塑造下层个体，个体也不能被当作完全独立样本。

一、核心直觉：个体不是漂浮在真空中的个体

多层模型，英文常叫 Hierarchical Linear Model，简称 HLM，也常被称为 Multilevel Model。它适合处理嵌套数据：学生嵌套在学校中，居民嵌套在社区中，企业嵌套在城市中，个人嵌套在国家中。它的底层判断是，同一上层单位内部的样本更相似，因此不能把所有样本都当作彼此独立。

一句话记住：普通回归只看“个人差异”，HLM 同时看“个人差异”和“环境差异”。

二、为什么需要 HLM：同一组内部的人往往更像

普通 OLS 回归默认每个观察值相互独立，但嵌套数据通常不满足这个条件。同一学校的学生会共享教师、校风和资源；同一社区的居民会共享空间环境、物业服务和治理能力；同一城市的企业会共享政策环境、产业基础和市场条件。

统计问题

标准误可能被低估

如果同一社区内居民高度相似，却被当作完全独立样本，模型会以为信息量比实际更多，显著性可能被夸大。

理论问题

环境影响会被遗漏

如果只研究个人特征，就会忽略社区治理能力、学校资源、城市政策环境等上层因素对个体结果的影响。

三、ICC：先看差异有多少来自上层单位

HLM 通常从空模型开始。空模型没有解释变量，只把结果变量的差异拆成两部分：组内差异和组间差异。ICC 就是组间差异占总差异的比例。

ICC = 组间方差 / (组间方差 + 组内方差)
ICC 越高，说明同一组内样本越相似，上层环境越重要。

比如研究居民政治参与，如果 ICC = 0.18，说明大约 18% 的参与差异来自社区之间的差异，而不是个人之间的差异。这就提示我们，不能只用个人层变量解释政治参与，还要考虑社区层因素。

四、随机截距模型：不同组可以有不同起点

随机截距模型允许每个上层单位拥有自己的基础水平。比如不同社区的平均参与意愿本来就不同，有的社区组织活跃，有的社区较为冷淡。随机截距就是承认这种“起点差异”。

第一层：Y_ij = β0j + β1 X_ij + r_ij
第二层：β0j = γ00 + u0j
合起来：Y_ij = γ00 + β1X_ij + u0j + r_ij

这里 i 表示个人，j 表示社区。u0j 表示第 j 个社区相对于总体平均水平的偏离。它不是误差，而是社区层面的随机差异。

五、随机斜率模型：同一个变量在不同组中的作用可以不同

随机斜率模型进一步认为，个人层变量的影响不一定在所有社区都一样。比如教育程度对政治参与的影响，可能在组织动员强的社区更强，在组织动员弱的社区更弱。

Y_ij = β0j + β1j X_ij + r_ij
β1j = γ10 + u1j

这里 β1j 表示 X 在第 j 个社区中的影响。随机斜率允许每个社区有自己的斜率，从而把“同一个个人特征为什么在不同环境中作用不同”纳入模型。

六、跨层互动：上层环境如何改变个人变量的作用

跨层互动是 HLM 最有解释力的部分。它关注上层变量 W 是否会改变个人层变量 X 对结果 Y 的影响。比如社区组织密度是否会增强居民政治兴趣对参与行为的影响。

个人层：Y_ij = β0j + β1j X_ij + r_ij
社区层：β1j = γ10 + γ11 W_j + u1j
γ11 就是跨层互动效应

如果 γ11 显著，说明个人层关系受到上层环境调节。它回答的不是“社区是否直接影响个人”，而是“社区环境是否改变个人特征的作用方式”。

七、中心化：比较的是“组内差异”还是“组间差异”

HLM 中经常要处理中心化。中心化不是纯技术操作，而是决定系数含义。总均值中心化是把变量减去总体平均值，组均值中心化是把变量减去本组平均值。

中心化方式含义适用理解
不中心化保留原始尺度截距有时难解释，跨层解释可能混杂
总均值中心化和总体平均水平比较适合解释整体偏离平均水平的作用
组均值中心化和本组平均水平比较适合识别组内效应，区分组内和组间差异

比如研究个人收入对参与的影响，组均值中心化后，收入系数表示“在同一个社区内，比本社区平均收入更高的人是否更愿意参与”。这和“富裕社区整体参与更高”不是同一个问题。

八、互动演示：ICC、随机截距和随机斜率

调整组间方差和组内方差。ICC 越高，说明差异越多来自组与组之间，越有必要考虑多层模型。

九、HLM 研究通常怎么做

第一步

确认层级结构

说明样本如何嵌套，比如居民—社区、学生—学校、企业—城市。

第二步

估计空模型和 ICC

先判断结果变量有多少差异来自上层单位，是否有必要使用多层模型。

第三步

逐步加入变量

先加个人层变量，再加上层变量，最后考虑随机斜率和跨层互动。

论文写作中，最好说明为什么普通回归不够，为什么数据具有嵌套结构，以及每一层变量对应什么理论含义。

十、常见误区：HLM 不是把变量分层摆放就结束

误区一：没有嵌套结构也使用 HLM

HLM 适合层级数据。如果样本没有清楚的上层单位，使用多层模型就缺少依据。

误区二：只看显著性，不看 ICC

ICC 是理解层级结构的重要起点。它告诉我们上层单位是否真的解释了结果差异。

误区三：随机斜率随便加

随机斜率需要理论理由，也需要足够的上层单位数量，否则模型可能不稳定。

误区四：中心化含义不清

不同中心化方式会改变系数解释。尤其在跨层互动中，中心化必须与研究问题一致。

十一、自测

问题 1：居民嵌套在社区中，为什么不能简单把所有居民当作完全独立样本？
因为同一社区居民共享社区环境，彼此更相似。忽略这种相关性会低估标准误，也会遗漏社区层影响。
问题 2：ICC=0.25 表示什么？
表示结果变量约 25% 的差异来自组间差异，说明上层单位具有重要影响。
问题 3：“社区组织密度增强政治兴趣对参与的影响”属于什么？
这是跨层互动。社区层变量调节了个人层变量对结果的影响。
一句话记住：HLM 把个体放回它所属的组织、社区和地区中，既看个人差异，也看环境差异。
