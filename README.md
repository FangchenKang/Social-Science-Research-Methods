# Social Science Research Methods

社会科学研究方法互动式学习导航。

本项目整理政治学、公共管理与社会科学研究中的常用方法讲义，将方法原理、学术脉络、适用场景、操作逻辑与常见误区汇集为可直接浏览的静态网站。

## 内容分类

- 统计与回归基础
- 因果推断
- 实验与调查设计
- 质性方法与方法论
- 文本、空间、网络与复杂方法

首页提供五条学习路径、全部方法的卡片式入口，以及按关键词、类别和难度筛选的目录。每份原有讲义均保留在根目录，并增加了返回首页入口；部分页面还补充了简明的学术历史脉络。

## 使用方式

- 本地直接打开 `index.html`。
- 或在 GitHub Pages 部署后通过网站地址访问。

本项目不依赖外部 CDN；样式与交互脚本均位于 `assets/`。
